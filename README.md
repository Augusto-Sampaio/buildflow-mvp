# 🚀 BuildFlow

BuildFlow is a modern construction project management dashboard designed to simplify project tracking, cost control, and decision-making in engineering and construction environments.

## 📌 About this Repository

This repository represents the **visual MVP and marketing version** of BuildFlow.

It was created to:
- Showcase the product concept
- Demonstrate UI/UX capabilities
- Present core features and workflows
- Serve as a foundation for the future SaaS platform

⚠️ This is **not the final SaaS architecture**. A new structured application will be developed separately.

---

## 🧠 Core Features (MVP)

- 📊 Project dashboard with progress tracking
- 💰 Cost management visualization
- 📈 Performance indicators (SPI / CPI)
- ⚠️ Risk and issue tracking (visual)
- 🧩 Modular UI components
- 📱 Responsive interface

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- Supabase (initial structure)
- Google AI (experimental features)
