# Construction Project Control Dashboard

A data-driven dashboard designed to monitor and control construction projects. The platform provides real-time insights into project performance, including schedule tracking, cost control, risk alerts and key performance indicators.

This project simulates a professional **project control environment used in engineering and construction management**, supporting decision-making through visual analytics and project KPIs.

---

# Live Demo

https://construction-project-control-dashbo.vercel.app

---

# Overview

Construction and engineering projects require continuous monitoring of **schedule, costs and performance indicators**. This system was developed to demonstrate how data-driven dashboards can support project managers and engineers in tracking project performance and identifying potential risks.

The application consolidates project data and presents key insights through interactive dashboards and visual analytics.

---

# Main Features

• Multi-project management dashboard  
• Planned vs Actual progress monitoring  
• Project performance indicators (SPI / CPI)  
• Cost control and budget deviation analysis  
• Cash flow monitoring  
• Project health and risk alerts  
• Individual project performance pages  
• Weekly performance analytics  
• Construction-focused KPIs and indicators  

---

# Tech Stack

• Next.js  
• React

• Supabase  
• Tailwind CSS

# Versão em Português

# Dashboard de Controle de Projetos de Construção

Um dashboard orientado a dados desenvolvido para monitorar e controlar projetos de engenharia e construção. A plataforma fornece insights em tempo real sobre o desempenho dos projetos, incluindo acompanhamento de cronograma, controle de custos, alertas de risco e indicadores de performance.

Este projeto simula um **ambiente profissional de controle de projetos utilizado em engenharia e gestão de obras**, apoiando a tomada de decisão por meio de análise de dados e visualização de indicadores.

---

# Demonstração

https://construction-project-control-dashbo.vercel.app

---

# Visão Geral

Projetos de engenharia e construção exigem acompanhamento constante de **cronograma, custos e indicadores de desempenho**. Este sistema foi desenvolvido para demonstrar como dashboards analíticos podem auxiliar gestores e engenheiros a acompanhar o desempenho dos projetos e identificar riscos antecipadamente.

A aplicação consolida dados de projetos e apresenta informações estratégicas através de dashboards interativos e indicadores de performance.

---

# Principais Funcionalidades

• Dashboard de gestão de múltiplos projetos  
• Acompanhamento de progresso planejado vs realizado  
• Indicadores de desempenho de projetos (SPI / CPI)  
• Controle de custos e análise de desvios orçamentários  
• Monitoramento de fluxo de caixa  
• Alertas de risco e saúde do projeto  
• Análise individual de desempenho por projeto  
• Análise semanal de performance  
• Indicadores voltados para gestão de obras
